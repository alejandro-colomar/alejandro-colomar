Prepare the RPi4 running ubuntu
===============================

Assuming a freshly installed Ubuntu server 20 in a RPi4, the following
script prepares the OS to be able to run this server.

- Install docker
- Install git

.. code-block:: BASH

	sudo ./bin/install.sh;
